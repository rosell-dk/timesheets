Initializers

An initializer sets up a convenient way to work with formulas

  • Convieniently create calculated fields - Just by setting a "data-formula" attribute...

Data types

In the library, we have added some datatypes and related functions. You can try them out in the following demos:

  • SimpleDate.js (work with dates)
  • Moment.js (for dates as well)
  • SimpleTime.js (work with times)
  • JQuery.js (work with jQuery objects)

Data type parsers

The parser parses numbers, strings and booleans per default. But its easily extended

  • How to create custom parsers - Shows how to make a "PI" parser. And a CELL parser

Bound variables (aka references)

A bound variable signals to the formula, when it changes

  • Element.js - Reference DOM elements and their various properties

Widgets

  • calculatedfield - Set custom formatter, and change formula of a field dynamically
  • editablefield - As above, but the field can be manually edited to override the calculation
